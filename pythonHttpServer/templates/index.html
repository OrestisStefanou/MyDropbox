What is my Dropbox?
MyDropbox provides you with 5GB free cloud storage!

How to use it:

1)Sign Up

2)Go to My Files

3)Select myDropboxApp to download the application

4)Extract the zip file in the directory you wish the myDropbox folder to be created

5)Run the command sudo ./myDropbox install to install the application

6)Run the command sudo ./MyDropbox start to start the application

7)Insert the files you wish to store in the cloud in your myDropbox folder

8)Run the command sudo ./MyDropbox stop if you wish to stop the application

You can access your files online by logging in and going to My Files
